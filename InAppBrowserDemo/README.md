# 移动设备 - 打开外链示例程序

In App Browser Demo for Meteor on Mobile Device


## 简介

- 支持Android和iOS
- 通过设备的系统浏览器，打开外链
- 使用包 http://plugins.cordova.io/#/package/org.apache.cordova.inappbrowser
