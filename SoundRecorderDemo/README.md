# 移动设备 - 录音机示例程序

Sound Recorder Demo for Meteor on Mobile Device


## 注意

- iOS录音文件会存储在 "/tmp/test1234.amr"
- Android录音文件会存储在 "/storage/sdcard0/test1234.amr"


## 简介

- 支持Android和iOS
- 使用.amr格式进行录音，文件尺寸非常小
- 使用包 https://github.com/luhuiguo/cordova-plugin-voice
